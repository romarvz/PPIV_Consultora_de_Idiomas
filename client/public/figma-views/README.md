# Vistas Estáticas para Figma

Esta carpeta contiene versiones estáticas de cada vista del sistema Lingua Academy, optimizadas para importar en Figma.

## Archivos incluidos:

- **landing.html** - Página de inicio con hero, funcionalidades y footer
- **dashboard.html** - Vista principal con métricas y alertas
- **students.html** - Gestión de estudiantes con tabla de datos
- **courses.html** - Catálogo de cursos en formato de tarjetas
- **reports.html** - Reportes con gráficos visuales (barras y circular)

## Características:

- ✅ Sin JavaScript interactivo
- ✅ CSS embebido para compatibilidad
- ✅ SVGs simplificados
- ✅ Estructura estática completa
- ✅ Navegación visual (sin funcionalidad)

## Cómo usar en Figma:

1. **Método directo**: Arrastra el archivo HTML a Figma
2. **Plugin recomendado**: Usa "HTML to Design" de Figma
3. **Alternativa**: Captura de pantalla del archivo abierto en navegador

## Notas:

- Cada archivo es independiente y completo
- Los colores y estilos son consistentes con el sistema original
- Optimizado para importación sin pérdida de diseño
